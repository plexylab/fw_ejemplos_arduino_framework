# Programas de prueba para Arduino, ESP32 y otras plataformas

Este repositorio está en desarrollo y reúne ejemplos básicos y prácticos para probar placas de desarrollo, módulos electrónicos, sensores, actuadores y periféricos de comunicación.

La finalidad de este repositorio es que cualquier usuario, incluso si está iniciando, pueda cargar un programa de prueba, realizar las conexiones necesarias y verificar rápidamente si su hardware funciona correctamente.

Actualmente se incluyen y se irán agregando ejemplos para diferentes plataformas, como:

* Arduino
* ESP32
* RP2040
* Raspberry Pi
* Otros microcontroladores y placas de desarrollo

Cada ejemplo busca ser simple, entendible y orientado a la acción, incluyendo comentarios en el código, configuración recomendada del IDE, pines de conexión y pasos básicos de prueba.

Este repositorio crecerá progresivamente con nuevos programas de prueba para facilitar la verificación de hardware antes de integrarlo en proyectos más grandes.
